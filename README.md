# AI Study Buddy

A premium, AI-powered study companion that helps you research topics, summarize articles, generate challenge questions, and create custom study plans.

## 🚀 Features
- **Topic Researcher**: Deep-dive into any educational subject.
- **Web-Search QA**: Get real-time answers from the internet.
- **Flashcard Quizzer**: interactive active recall practice.
- **Smart Summarizer 2.0**: Summarize text or web links.
- **Study Planner**: Custom Markdown study schedules.
- **Question Generator**: Challenge yourself with AI questions.
- **AI Tutor (QA)**: Extract answers from specific paragraphs.

## 🛠️ Installation

1. **Clone the project** and navigate to the directory.
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Set up API Key**:
   Ensure your Gemini API Key is set in `config.py`. (Already configured).

## 🏃 Running the App

Start the Flask server:
```bash
python app.py
```

Open your browser and go to:
[http://127.0.0.1:5000](http://127.0.0.1:5000)

## 🎨 Technologies
- **Backend**: Flask, Python
- **AI**: Google Gemini API, Hugging Face Transformers
- **Frontend**: Vanilla HTML5/CSS3 (Modern Glassmorphism)
